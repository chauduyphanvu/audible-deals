How global defaults work

Values set here act as fallback defaults for all deals commands. They override built-in CLI defaults, but are in turn overridden by any flag you pass on the command line or in a profile. For example, setting max-price = 5 means every deals find run will default to $5 max, unless you pass --max-price explicitly.
